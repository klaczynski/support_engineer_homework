REPORTS

  • Display Optimization Settings: Takes the value of a myShopifyDomain field as an input and returns their optimization settings.
  • Display All Organizations: Loops through all organizations and shows the date they were created (DD/MM/YYYY), their status, and planName sorted by oldest to newest.
  • Display Cancelled Organizations: Returns the list of organizations whose status is cancelled.
  • Display 1 Organization: Takes the value of an orgName and returns the organization record in JSON format.
